# Vehicle Rental System 🚗

A complete Vehicle Rental System developed using MySQL to manage vehicle rentals efficiently. This project automates rental operations by maintaining records of vehicles, customers, employees, rentals, returns, and branch details. It demonstrates practical implementation of database management concepts using SQL.

## 🚀 Features

* Vehicle Management System
* Customer Registration & Tracking
* Employee Management
* Rental & Return Tracking
* Billing Management
* Branch Management
* CRUD Operations
* SQL Joins & Subqueries
* Advanced Query Handling

## 🛠️ Technologies Used

* MySQL
* SQL Queries
* Relational Database Management System (RDBMS)

## 📊 Database Modules

### 1. Branch Management

Stores branch addresses and contact information.

### 2. Customer Management

Maintains customer details including:

* Name
* Address
* License Number
* Registration Date

### 3. Employee Management

Handles:

* Employee Details
* Positions
* Salaries
* Branch Allocation

### 4. Vehicle Management

Tracks:

* Vehicle Name
* Vehicle Type
* Rental Price
* Availability Status

### 5. Rental Management

Records vehicle rental transactions between customers and employees.

### 6. Return Details

Stores return dates and total billing information.
